# DevFlow Landing Page

Simple static landing page. Deploy anywhere.

## Deploy

**Vercel**
```bash
cd landing && vercel
```

**Netlify**
Drag the `landing` folder to netlify.com/drop

**GitHub Pages**
1. Enable Pages in repo settings
2. Source: Deploy from branch
3. Folder: `/landing` or root with `landing/index.html`

**Custom domain**
Point your domain to the deployment. Update GitHub/npm links in `index.html` if needed.
